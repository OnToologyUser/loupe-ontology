commonInstancesCountdp back to ToC or Data Property ToC

IRI: http://ont-loupe.linkeddata.es/def/core#commonInstancesCount

The total count of common instances of given two classes in a collection of triples. The count is calculated using the explicit triples materialized in the collection without any inferencing. (e.g., if there is an instance x that is defined as "x is an instance of Class A" and "x is an instance of Class B", then x is counted as a common instance for class A and B.)
has domain
CommonInstancesPartition c
has range
long

entityCountdp back to ToC or Data Property ToC

IRI: http://ont-loupe.linkeddata.es/def/core#entityCount

The total number of distinct entities in a given collection of RDF triples. Entities are defined in the RDF 1.1 Concepts and Abstract Syntax W3C Recommendation as any IRI or literal that denotes something in the world. Anything can be an entity, including physical things, documents, abstract concepts, numbers and strings.
has sub-properties
classCount dp, instance count dp, labelledEntityCount dp, literalCount dp, propertyCount dp, typedEntityCount dp

long
